# Terminal Training Git Project
## About the project
This repository contains training tasks for terminal, Git, and GitHub.

## Goal
The goal of this project is to build strong practical skills in terminal workflow, Git, and GitHub.

## Repository structure
- data/ — files for training data
- data/raw/ — raw data files
- data/processed/ — processed data files
- docs/ — project notes and documentation
- sql/ — SQL files for Git practice
- README.md — project overview
- .gitignore — ignored files and folders

## Skills demonstrated
- terminal navigation
- file and folder management
- Git tracking and commits
- GitHub repository organization
- README writing
- project structure design

## Status
This project is in progress and used as part of step-by-step engineering training.
